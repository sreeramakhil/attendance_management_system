## 📜 Open Source License (MIT)

### What is MIT License?

The **MIT License** is one of the most permissive open-source licenses. It allows almost complete freedom with very few conditions.

### MIT License Terms

#### ✅ You CAN:
- **Use commercially** - Use in production, commercial products
- **Modify** - Change, adapt, and build upon the code
- **Distribute** - Redistribute the software
- **Use privately** - For personal projects
- **Sublicense** - Include in your own projects with your license
- **Patent use** - Any patent claims are granted

#### ❌ You CANNOT:
- **Hold the author liable** - No warranty, no liability
- **Claim warranty** - Software provided "as is"

#### ⚠️ You MUST:
- **Include the original license** - Copy the LICENSE file
- **Include copyright notice** - Keep the original copyright notice
- **State changes** - Clearly mark modifications

If you use this project in your own:

## 💼 Commercial License

### When Do You Need a Commercial License?

You might need a commercial license if:

- ✅ You want to use the code **without the MIT attribution requirement**
- ✅ You want **warranty or liability protection**
- ✅ You need **custom terms or modifications**
- ✅ You want **exclusive usage rights**
- ✅ You require **SLA or support guarantees**
- ✅ Your usage conflicts with the MIT License conditions

### Commercial License Options

#### **Option 1: Standard Commercial License**
- Single commercial project license
- No attribution required
- Custom terms negotiable
- **Cost**: Contact for pricing

#### **Option 2: Enterprise License**
- Multiple projects
- Full source code access
- Priority support
- Custom features/modifications
- **Cost**: Contact for pricing

#### **Option 3: White-Label License**
- Rebrand as your own
- Remove original attribution
- Full customization rights
- **Cost**: Contact for pricing

## 📝 How to Get a License

### Step 1: MIT License (Automatic - No Action Needed)
The MIT License is **already applied** to this project:

1. Clone the repository
   ```bash
   git clone https://github.com/sreeramakhil/attendance_management_system.git
   ```

2. Include the [LICENSE](LICENSE) file in your project
   ```bash
   cp LICENSE your-project/
   ```

3. Add attribution in your README:

   - [Attendance Management System](https://github.com/sreeramakhil/attendance_management_system)

## ❓ License FAQ

### Q1: Can I use this project commercially with the MIT License?
**A:** Yes! The MIT License explicitly allows commercial use. You just need to include the original LICENSE file and copyright notice.

### Q2: Do I need to open-source my project if I use this?
**A:** No. The MIT License does NOT require you to open-source your project. You can use it in commercial, closed-source projects.

### Q3: Can I modify the code under MIT License?
**A:** Yes. You can modify, adapt, and build upon the code. You just need to include the original license notice.

### Q4: What's the difference between MIT and GPL licenses?
**A:** 
- **MIT**: Permissive, allows closed-source use
- **GPL**: Copyleft, requires derived works to be open-source

### Q5: Can I remove the copyright notice?
**A:** No. The MIT License requires you to include the original copyright notice. You must keep: "Copyright (c) 2026 Sreeram Akhil"

### Q6: Do I get support with the MIT License?
**A:** The MIT License includes no warranty or support. For production deployments, consider a commercial license with SLA.

### Q7: Can I charge money for a project using this?
**A:** Yes, absolutely! You can sell software that uses this code. Just include the LICENSE file.

### Q8: What if I don't comply with the MIT License?
**A:** Non-compliance may expose you to legal action for copyright infringement. Always include proper attribution and the original LICENSE file.

### Q9: How do I add this to my project?
**A:** 
```bash
# Copy the LICENSE file
cp LICENSE your-project/LICENSE-AttendanceSystem

# Add to your README
echo "This project includes components from Attendance Management System (MIT License)" >> your-project/README.md
```

### Q10: Can I use this in a proprietary product?
**A:** Yes! The MIT License explicitly allows proprietary use. Just include the LICENSE notice.
